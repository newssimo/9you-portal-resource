# 9you-portal-resource

本仓库用于归档和发布多个独立 HTML 页面。这些页面均为静态资源，不针对任何特定域名或具体网站，仅作为通用前端页面集合进行维护。

## 项目简介

`9you-portal-resource` 是一个静态 HTML 页面归档仓库。仓库中的页面以独立、自包含的方式存放，可用于快速部署、演示或作为前端开发参考。所有页面均不包含后端逻辑或动态数据依赖。

## 目录结构

```
9you-portal-resource/
├── pages/          # 存放独立 HTML 页面文件（按功能或场景分类）
├── assets/         # 公共静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- `pages/`：每个子目录或文件代表一个独立的页面归档。页面文件名或目录名可反映其用途或类型。
- `assets/`：存放页面共享的样式、脚本及媒体资源，便于统一管理。

## 页面归档说明

- 所有页面均为纯 HTML 文件，可直接在浏览器中打开运行。
- 页面之间相互独立，无内部链接跳转或依赖关系。
- 归档内容可能随时间更新或清理，不保证所有页面长期可用。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/9you-portal-resource.git
   ```
2. 直接使用浏览器打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可将整个仓库部署至任意静态文件服务器或 CDN 进行访问。
